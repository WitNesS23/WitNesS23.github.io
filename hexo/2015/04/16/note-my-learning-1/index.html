关于整体布局架构

拿到这个作业，首先先对各个布局做简单的分析。

  • 三部分组成，首部为一个简单标题todos；
  • 中间主体部分包含一个输入区域、显示区域和一个统计区域；
  • 尾部为一个用法描述和编辑人名称；

感觉这个显而易见啊。 ( / ω ╲ )

关于不熟悉的部分

CSS3中相关的阴影操作

在Demo中标题todos，主体统计区域，Clear completed按钮都使用到了阴影的操作。查看老师的编写源代码，主要是CSS3 的两个新属性 box-shadow和rgba()的配合使用。

其中rgba()相对容易掌握，但是box-shadow的熟练使用还是需要自己钻研。

伪类选择器和伪元素选择器的使用
